# Helldivers-2-Randomizer
As the title implies, this is a loadout randomizer for Helldivers 2
I wrote it at the request of some friends who play the game.
It takes a number of csv files and generates weighted lists for all the equipment, weapons, strategmens, etc. and spits out a loadout that can be used in-game.
It has not been updated in some time but if you wish to update the lists or make some improvements to the code, fill free to put in a pull request.
